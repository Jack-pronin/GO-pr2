## Описание проекта

Минимальный Go-сервер с правильной структурой директорий:
- `cmd/myapp/` — входной файл main.go
- `internal/app/` — логика приложения (сервер, маршруты)
- `utils/` — вспомогательные утилиты (логгер, JSON-ответы)
